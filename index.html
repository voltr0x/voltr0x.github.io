Things I've developed

03. PROJECTS

VR Two-Gether

A cross-platform two player cooperative game where players collaborate to solve puzzles based on cooperative design patterns. The game features balanced gameplay mechanics and VR/PC affordances to create meaningful cooperation between players and determine player preferences for entertainment and performance through each level's new mechanics.

2D Game Engine

As a final project for my game engines class, I was part of a group project which required us to build a game engine using C++, SDL2 and OpenGL. We also created 3 games using this engine - breakout game, a platformer and a 2D top down adventure dungeon crawler RPG game. This project took one week to complete, with an estimated development time of 40 hours, during which time I implemented input processing, 2D rendering, a collision system, and rudimentary physics on the game objects.

Pong Clone

As a final project for my graphics class, I enjoyed creating an old-school classic game using C++ and SDL2. This project took one week to complete, with an estimated development time of 20 hours, during which time I implemented input processing, 2D rendering, a collision system, and rudimentary physics on the game objects.

Shoot the Zombies!

A top down isometric view survival shooter where players have to shoot waves of zombies. The assets are developed on Blender which I have created for the first time, whereas the game is developed on Unity with implemented game play mechanics of levels (increasing difficulty with each wave of enemies) and power-ups such as ammo and health.

Melon Smash

A group of 5 students from all over the world collaborated to create a single-player tower defense game in Unity. The game revolves around a farmer attempting to preserve his prized pumpkin by tossing several types of melons, one of which deals damage and the other of which is used to lure enemies. We divided tasks according to our areas of interest; I was in charge of opponent AI, gameplay, level design, and game management.

Chrysalis

Faculty-Driven Game Studio Project. Chrysalis: The Transformation lands uses the immersive nature of virtual reality (VR) to support patients through guided physical rehabilitation. Using principles of game science and design Chrysalis aims to support the long-term engagement of patients with their rehabilitation protocols. I am working on the VR avatar and implementing different types of exercises with the help of trackers.

3D Object Model Parser

Parse and render .obj files with vertex, texture and normal data with help of 3D math using C++ and OpenGL as one of the assinments in my computer graphics class.
